About the Project

This HR Analytics project focuses on analyzing employee attendance, work-from-home (WFH), and leave trends using Power BI. By transforming and visualizing attendance data, the dashboard helps identify workforce patterns and supports data-driven HR decision-making.

Project Description
1) Built an interactive HR Attendance Analytics dashboard using Power BI to analyze employee attendance, WFH, and leave trends from 1,000+ attendance records.
2) Cleaned, transformed, and standardized attendance data using Power Query to create a reliable dataset for analysis and reporting.
3) Developed DAX measures to track key HR metrics, including WFH (11.15%) and Sick Leave (2.01%), with analysis by weekday and month.
4) Created dynamic visualizations and filters to enable easy exploration of attendance and leave patterns across different time periods.
5) Generated actionable workforce insights to support HR planning, attendance monitoring, and data-driven decision-making.

Tools Used
1) MS Excel
2) MySQL
3) Power BI
4) Power Query
5) DAX

Key Insights
1) Identified overall Work From Home (WFH) rate of 11.15%.
2) Analyzed Sick Leave (SL) utilization rate of 2.01%.
3) Tracked attendance, WFH, and leave trends by weekday and month.
4) Highlighted workforce availability patterns to support HR planning.
5) Enabled quick monitoring of attendance metrics through interactive dashboards.
